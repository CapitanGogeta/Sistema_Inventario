# Roadmap y Pendientes — Sistema Inventario Hildemar

**Fecha Actualizada:** 29/04/2026

## 1. Sistema de Ventas y Finanzas
- [ ] **Aumento de precio por horario (Horario nocturno):** Lógica para aumentar precios automáticamente fuera de horario, respetando el precio original a clientes que ya estaban consumiendo.
- [ ] **Descuentos a tickets:** Capacidad de aplicar descuentos manuales al total del ticket para clientes especiales.
- [ ] **Estandarización de precios (Dólares):** Validar que los precios en BD sean estrictamente en USD y resolver discrepancias de productos cargados en Bs.

## 2. Reportes y Dashboard
- [ ] **Dashboard Avanzado:** Implementar gráficos visuales (Chart.js) para "Productos más vendidos" y estadísticas financieras.

## 3. Infraestructura y Servicios
- [ ] **Servicio de Emails:** Sistema de alertas y recuperación de contraseñas.

## 4. Frontend y UX
- [ ] **Lectura por Código de Barras:** Lógica para descontar productos directamente escaneando el código de barras sin hacer clics.

## 5. Mejoras de Trazabilidad
- [ ] **Auditoría de Precios:** Registro histórico de cambios de precio para saber cuándo y quién subió el costo de un producto.
